
# Hi Web Task

in this project we have a list of products & a login page To authenticate user

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run dev
```

## Tech Stack
- React
- TailwindCSS
- Typescript

## Libraries
- react-modal
- react-redux
- axios
- react-router-dom
- react-toastify
- @reduxjs/toolkit
